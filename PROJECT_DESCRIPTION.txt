================================================================================
                              PROJECT OVERVIEW
================================================================================

This is a modern, responsive personal portfolio website built for Vaibhav Raj,
a Computer Science student and Web Developer. The portfolio showcases his skills,
projects, services, and provides a contact form for potential clients/employers.

The project is built using React.js with Vite as the build tool and uses 
Tailwind CSS for styling. It features smooth animations powered by Framer Motion,
a custom cursor effect, and EmailJS integration for the contact form.

Key Features:
- Modern dark-themed design with glassmorphism effects
- Smooth scroll-triggered animations
- Custom animated cursor
- Interactive project showcase with hover effects
- Working contact form with EmailJS integration
- Fully responsive design (mobile-friendly)
- Route-based navigation for project details

6. main.jsx

   Purpose: React application entry point
   
   Function:

   - Renders App component wrapped in StrictMode

7. App.jsx
   --------
   Purpose: Main application component and routing configuration
   
   Features:

   - Defines two routes:

     * "/project/:id" - Individual project details page

   b) skills - Object with skill categories:

      - tools: ["MongoDB", "MySQL", "Git", "GitHub", etc.]

================================================================================
                              HOW IT WORKS
================================================================================

Application Flow:
-----------------

1. User visits the website
   → index.html loads
   → main.jsx initializes React
   → App.jsx renders with Router

2. Home Route ("/"):
   → Home component renders all sections in order:
      Navbar → Hero → Skills → Services → Projects → Process → Contact → Footer
   → Custom Cursor follows mouse
   → Animations trigger as user scrolls (whileInView)

3. Project Detail Route ("/project/:id"):
   → ProjectDetails component renders
   → Fetches project data from data.js by ID
   → Shows full project information

4. Contact Form:
   → User fills out form
   → On submit, EmailJS sends email to vaibhav.work5860@gmail.com
   → Form shows success/error feedback


Key Technologies:
-----------------

1. React 19 - UI library with hooks
2. Vite - Fast build tool with HMR
3. Tailwind CSS 4 - Utility-first CSS framework
4. Framer Motion - Animation library
5. React Router 7 - Client-side routing
6. EmailJS - Email service integration
7. Lucide React - Icon library


Styling Approach:
-----------------

- Dark theme (black background, white/gray text)
- Glassmorphism effects (backdrop blur, subtle borders)
- Grid-based layouts
- Responsive design (mobile-first with md/lg breakpoints)
- Custom fonts: Syne (display), Inter (body)
- Subtle noise texture overlay
- Custom cursor replacing system cursor
